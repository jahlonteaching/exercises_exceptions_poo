# Lista de ejercicios Excepciones y POO

## Ejercicio 1
Crea una clase en Python llamada `Shape` con un método `area()` y `perimeter()`. Implementa las clases `Rectangle`, `Square` y `Circle` que hereden de la clase `Shape` y anulen los métodos `area()` y `perimeter()` según corresponda.

## Ejercicio 2
Escribe una clase en Python llamada `Fraction` que tenga atributos `numerator` y `denominator`. Implementa métodos para sumar, restar, multiplicar y dividir fracciones. Incluye manejo de excepciones para casos en los que el denominador es cero.

## Ejercicio 3
Escribe una clase en Python llamada `Matrix` que tenga atributos `rows` y `columns`. Implementa métodos para sumar, restar y multiplicar matrices. Incluye manejo de excepciones para casos en los que se pasen matrices con dimensiones incompatibles a los métodos.

## Ejercicio 4
Escribe una clase en Python llamada `Temperature` que tenga atributos `value` y `unit` (Celsius, Fahrenheit o Kelvin). Implementa métodos para convertir entre unidades y comparar temperaturas. Incluye manejo de excepciones para casos en los que se pase una unidad no válida.

## Ejercicio 5
Escribe una clase en Python llamada `PatternMatcher` que tenga un método `match` que tome una lista de cadenas y una cadena de patrón, y devuelva una lista de cadenas que coinciden con el patrón. Incluye manejo de excepciones para casos en los que se pase una cadena de patrón no válida.

## Ejercicio 6
Escribe una clase en Python llamada `Statistics` que tenga un método `median` que tome una lista de números y devuelva el valor mediano, y un método `standard_deviation` que tome una lista de números y devuelva la desviación estándar. Incluye manejo de excepciones para casos en los que la lista esté vacía o contenga valores no numéricos.

## Ejercicio 7
Escribe una clase en Python llamada `FileSearcher` que tenga un método `search` que tome una lista de nombres de archivo y una cadena de búsqueda, y devuelva una lista de líneas coincidentes. Incluye manejo de excepciones para casos en los que no se encuentre un archivo o no se pueda leer.

## Ejercicio 8
Escribe una clase en Python llamada `ProductFinder` que tenga un método `largest_product` que tome una lista de enteros y devuelva el producto más grande que se puede obtener multiplicando cualquier tres enteros. Incluye manejo de excepciones para casos en los que la lista sea demasiado corta o contenga valores no numéricos.

## Ejercicio 9
Escribe una clase en Python llamada `SumFinder` que tenga un método `find_pairs` que tome una lista de números y devuelva una lista de todas las parejas que sumen un valor objetivo. Incluye manejo de excepciones para casos en los que la lista sea demasiado corta o contenga valores no numéricos.
